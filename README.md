# Hypnagogic 

An iOS app to play with [Nannou](https://nannou.cc). This requires a bridge between
Rust code and Swift.

## Setting up your env

Ensure you have Rust installed (through [rustup](https://rustup.rs)) and run:

``` sh
./setup-env.sh
```
